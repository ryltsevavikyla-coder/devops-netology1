\## Описание .gitignore



Добавлен файл `.gitignore`, чтобы Git игнорировал временные и служебные файлы.



В папке `terraform/` будет игнорироваться:

\- директория `.terraform/` с кэшем

\- файлы состояний `\*.tfstate`

\- логи ошибок `crash.log`

\- override файлы конфигурации

\- и другие ненужные файлы Terraform.
