## Выполнено:

- Создан каталог `branching` с файлами `merge.sh` и `rebase.sh`
- Создана ветка `git-merge` с изменениями скрипта ( @ вместо * и use shift )
- Создана ветка `git-rebase` с изменениями скрипта
- Выполнен merge ветки `git-merge` в `main`
- Выполнен rebase ветки `git-rebase` на `main` (с разрешением конфликтов)
- Force push и финальный merge
